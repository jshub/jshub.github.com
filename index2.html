jsHub helps eliminate web page tagging overload and data duplication. We make tools to make web site tagging easy, fast and secure. We host open standards to reduce data inconsistencies across vendors.


  • Welcome to jsHub

    It is common for web pages to contain many different tracking tags used for web analytics, advertising tracking, targetting, audience measurement, and more. Some sites have more than 10 tags per page.

    Each tag adds either HTML or JavaScript to the page adding to its weight: more tags means larger pages which means longer load times. And each tag ends up reading the same information from the page, such as its name, which can lead to inconsistencies.

    A fresh approach

    jsHub is a suite of tools and a set of standards that allows you to declare data on your site once, and use it to power all the tag-based products you use.

  • Standard code

    The jsHub tag is written in JavaScript and distributed under the BSD license.

    It uses plugins to provide consistent data to multiple products.

    Comes with tests

    If you are a JavaScript developer, come and see how we do fully automated, cross-browser, continuous integration testing.

  • Standard data

    To ensure that data is only declared once we propose a page markup standard based on HTML microformats, and released into the public domain, for common page metadata. 

            
    
          Homepage

    Implementing metadata means writing HTML, not JavaScript code.

    Because the data appears once, implementation is no longer an exercise in repeating yourself. And because it is an open standard, it does not lock you in to a single "universal tag" provider.

    Comes with best practice

    The demo store shows the tag, markup and tag inspector working together. Giving you a real life example of how to deploy the code.

  • Website owners

    If you run a site, jsHub can help you with consistent information across different products, lower tag weight and the assurance that the tag is widely tested and secure.

    Comes with tools

    Using our Tag Inspector, you can self-service and check what is actually being sent from your tag implementation.

    End users benefit from faster page load times and greater transparency. With the Tag Inspector an end user can quickly see what information is being gathered on the page.

  • Use jsHub to power your product

    If your product uses tags to collect data from pages, then it is likely that your customers are spending time to reconcile your numbers with other products.

    Cross product data problems are eliminated as data is read once. Your customers can use common code to remove most of their tag maintenance, and tag installation becomes easier as jsHub becomes widely adopted.

    See the standards pages for more on why unifying markup and tags helps your product and your customers.
